The first program reads a valid arithmetic operations. 
Assume the arithmetic operation sign for the purposes of the description to be of a 
(+, -, *, /). They can be in the form of a & b with or without spaces inbetween.

The second program reads two valid arithmetic operations. 
Assume the arithmetic operation sign for the purposes of the description to be of a
(+, -, *, /). They can be in the form of a & b & c with or without spaces inbetween.
